# HRM SYSTEM – FUNCTIONAL DOCUMENT

## 1. Thông tin chung

| Mục              | Nội dung                                     |
| ---------------- | -------------------------------------------- |
| **Tên hệ thống** | Human Resource Management (HRM)              |
| **Đối tượng**    | Doanh nghiệp                                 |
| **Role**         | Sếp, Nhân viên                               |
| **Mục tiêu**     | Quản lý nhân sự, chấm công, nghỉ phép, lương |

## 2. Danh sách Role

| Role          | Mô tả                             |
| ------------- | --------------------------------- |
| **Sếp**       | Quản trị hệ thống, duyệt, báo cáo |
| **Nhân viên** | Sử dụng các chức năng cá nhân     |

## 3. Chức năng chung

| STT | Nhóm chức năng | Chức năng     | Mô tả                    |
| --- | -------------- | ------------- | ------------------------ |
| 1   | Xác thực       | Đăng nhập     | Đăng nhập hệ thống       |
| 2   | Xác thực       | Quên mật khẩu | Reset mật khẩu qua email |
| 3   | Tài khoản      | Đổi mật khẩu  | Đổi mật khẩu cá nhân     |
| 4   | Hồ sơ          | Xem hồ sơ     | Xem thông tin cá nhân    |
| 5   | Thông báo      | Xem thông báo | Thông báo hệ thống       |

## 4. Chức năng Nhân viên

| STT | Chức năng            | Mô tả                  |
| --- | -------------------- | ---------------------- |
| 1   | Check-in / Check-out | Ghi nhận giờ vào ra    |
| 2   | Xem lịch sử công     | Xem chấm công cá nhân  |
| 3   | Tạo đơn nghỉ phép    | Gửi yêu cầu nghỉ       |
| 4   | Xem lương            | Xem bảng lương cá nhân |

## 5. Chức năng Sếp

| STT | Chức năng         | Mô tả                     |
| --- | ----------------- | ------------------------- |
| 1   | Quản lý nhân viên | Thêm, sửa, khóa nhân viên |
| 2   | Duyệt nghỉ phép   | Approve / Reject đơn      |
| 3   | Quản lý chấm công | Xem và chỉnh sửa công     |
| 4   | Quản lý lương     | Tạo và duyệt bảng lương   |
| 5   | Báo cáo           | Dashboard & thống kê      |
